# shop_project
Simple E-commerce shop project. 

What is needed to start:

Install:

  Python libraries:
    -django
    -django-rest-framework
    -django-cors-headers
    -djang-filter
    -djoser
    -pillow

  Node.js libraries:
    -bulma
    
Activate venv: \project: env/scripts/activate.ps1

Run server:
\project\shop_project: python manage.py runserver

\project/shop_vue: npm run serve
